Instructions

Welcome to Minecraft 2D Regressed! In this 2D version you can build your own world with two dimensional tiles.

  1. The user will have 3 types of tools:
    1. Axe - for cutting trees
    2. Pickaxe - for mining rocks
    3. Shovel - for digging dirt
  2. Clicking on a tool followed by clicking on a tile in the world will remove the tile. (If it is of the correct type according to #1) and add it to the inventory
  3. User can click on the tile in the inventory and place it back in the world (just the last one)
